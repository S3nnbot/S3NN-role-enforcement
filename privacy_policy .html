Privacy Policy for S3NN

Last updated: June 7, 2026

1. Introduction

S3NN ("the Bot") is a Discord bot that manages role assignments within Discord servers. This Privacy Policy explains what data the Bot collects, how it is used, and how it is stored.

2. Data We Collect

The Bot collects and stores the following data solely for the purpose of providing its functionality:

  • Server (Guild) IDs — to store per-server configuration.
  • Role names — as configured by server administrators via slash commands.
  • Channel IDs — only if a server administrator sets a log channel using /setlog.

The Bot does not collect or store any of the following:

  • User IDs or usernames
  • Message content
  • Personal information of any kind

3. How Data Is Used

Collected data is used exclusively to provide the Bot's role enforcement functionality. Configuration data is used to determine which roles belong to which paths within a server and to send log messages to designated channels.

4. Data Storage

All data is stored locally in a JSON file on the server hosting the Bot. Data is not shared with any third parties.

5. Data Retention

Server configuration data is retained until a server administrator uses the /reset command to delete it, or until the Bot is removed from the server.

6. Your Rights

Server administrators may delete all stored data for their server at any time using the /reset command.

7. Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of the Bot following any changes constitutes acceptance of the new policy.

8. Contact

If you have any questions about this Privacy Policy, please contact the Bot developer via the support server linked in the Bot's profile.
